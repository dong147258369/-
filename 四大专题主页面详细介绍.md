## 二、页面结构分析

### 2.1 HTML结构

页面采用标准的HTML5语义化结构，主要分为以下几个部分：

**1. 头部区域 (header)**
   - 显示品牌名称"国筑万象"
   - 副标题展示四大类型：民居、官府、皇宫、桥梁
   - 采用古建筑风格配色（朱红+金黄）
   - 金色双线边框装饰

**2. 内容区域 (main)**
   - **narrative-intro**：介绍文字区域，深色半透明背景
   - **topic-tree-container**：树形图容器，650px高度
   - **topic-modal**：建筑详情悬浮窗（默认隐藏）

**3. 悬浮窗结构**
   - 模态框背景（点击关闭）
   - 内容区：左侧图片+文字介绍，右侧3D词云
   - 关闭按钮

### 3.3 ECharts树形图配置详解

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 布局类型 | radial | 径向布局，以圆心为中心向外辐射 |
| 排列方向 | TB | 从上到下，根节点在上 |
| 连线形状 | curve | 平滑曲线连接节点 |
| 初始展开层级 | 2 | 默认展开到建筑列表层级 |
| 交互功能 | roam: true | 支持拖拽和缩放 |
| 动画效果 | cubicOut | 平滑的展开/收起动画 |
| 动画时长 | 1200ms | 初始加载动画 |

**关键配置代码：**

```javascript
series: [{
  type: 'tree',
  layout: 'radial',           // 径向布局
  orient: 'TB',                // 从上到下
  edgeShape: 'curve',          // 曲线连接
  initialTreeDepth: 2,         // 默认展开2层
  roam: true,                  // 支持拖拽缩放
  expandAndCollapse: true,     // 可展开收起
  
  symbolSize: function(value, params) {
    if (params.data.name === '国筑万象') return [36, 36];
    if (params.data.symbol && params.data.symbol.startsWith('image://')) 
      return [50, 50];
    return params.data.children ? [24, 24] : [14, 14];
  },
  
  itemStyle: {
    color: 'rgba(139, 69, 19, 0.3)',
    borderColor: 'rgba(218, 165, 32, 0.5)',
    borderWidth: 1,
    shadowBlur: 8,
    shadowColor: 'rgba(218, 165, 32, 0.2)'
  }
}]
```

## 四、样式设计详解

### 4.1 背景图片设置

**topics.html专用样式（内联）：**

```html
<style>
  body {
    background: url('images/detailed_images/background.png') 
                no-repeat center center fixed !important;
    background-size: cover !important;
  }
</style>
```

**特点：**
- 全屏覆盖，无重复
- 中心定位，随滚动固定
- 使用!important确保优先级最高
- 覆盖全局CSS中的背景设置

### 4.2 色彩方案

**专题分类颜色：**

| 类型 | 颜色值 | 颜色名称 | 建筑含义 |

## 五、交互功能详解

### 5.1 悬浮窗系统

**触发方式：**
- 点击任意建筑节点（叶子节点）

**显示内容：**
1. **左侧区域**
   - 建筑图片（如有）
   - 建筑名称
   - 位置和类型信息
   - 详细介绍文字

2. **右侧区域**
   - 3D球形词云
   - 展示该建筑的工艺关键词
   - 如：斗拱、榫卯、琉璃瓦等

3. **底部操作**
   - "查看详情"链接按钮
   - 跳转至 detail.html 页面

**关闭方式：**
- 点击关闭按钮（×）
- 点击模态框背景
- 按ESC键

**代码实现：**

```javascript
function showBuildingModal(building) {
  // 设置建筑图片
  if (building.imageUrl) {
    imgEl.innerHTML = `<img src="${building.imageUrl}" alt="${building.name}" />`;
  }
  
  // 设置文字介绍
  subtitleEl.textContent = building.subtype + ' · ' + building.location;
  introEl.textContent = building.feature || building.craft;
  
  // 生成词云
  const keywords = getBuildingKeywords(building);
  tagCloudInstance = TagCloud(container, keywords, {
    radius: 120,
    maxSpeed: 'normal'
  });
  
  // 显示悬浮窗
  modal.classList.add('is-open');
}
```

### 5.2 树形图交互

| 操作 | 效果 |
|------|------|
| 点击专题图标 | 展开/收起该分类下的建筑列表 |
| 点击建筑名称 | 打开详情悬浮窗 |
| 鼠标拖拽空白处 | 整体移动树形图位置 |
| 滚轮上下滚动 | 放大/缩小视图 |
| 鼠标悬停节点 | 显示节点提示信息（Tooltip） |

**Tooltip提示格式：**

## 七、性能优化措施

| 优化项 | 实现方式 | 效果 |
|--------|----------|------|
| 懒加载 | 建筑图片仅在悬浮窗打开时加载 | 减少初始请求 |
| 实例复用 | ECharts实例dispose后重建 | 避免内存泄漏 |
| 动画优化 | CSS3动画 + requestAnimationFrame | 流畅60fps |
| 事件委托 | ECharts原生事件系统 | 减少监听器数量 |
| 代码分割 | 按页面拆分JS文件 | 按需加载 |

## 八、使用说明

### 8.1 查看建筑详情步骤

1. 打开四大专题页面（访问 topics.html）
2. 等待树形图加载完成（约1-2秒）
3. 点击四大分类图标（民居/官府/皇宫/桥梁）展开建筑列表
4. 点击具体建筑名称节点
5. 在悬浮窗中查看建筑介绍、图片和工艺词云
6. 点击"查看详情 →"跳转到独立详情页面

### 8.2 导航操作指南

- **拖拽空白处**：移动整个树形图，查看不同区域
- **滚轮滚动**：放大/缩小视图，聚焦细节或查看全貌
- **点击专题图标**：展开或收起该分类下的所有建筑
- **ESC键**：快速关闭悬浮窗

## 九、二次开发指南

### 9.1 添加新建筑数据

在 js/data.js 中找到对应类型的数据数组，添加新对象：

```javascript
const MINJU_DATA = [
  // ... 现有建筑数据
  {
    name: '新建筑名称',
    location: '所在地点',
    province: '省份',
    era: '朝代',
    subtype: '建筑子类',
    feature: '建筑特色描述',
    craft: '工艺1、工艺2、工艺3',
    imageUrl: 'images/detailed_images/新图片.jpg'
  }
];
```

### 9.2 修改节点样式

编辑 js/topics.js 中的配置：

**修改形状：**
```javascript
const TYPE_SHAPES = {
  minju: 'circle',      // 改为圆形
  guanfu: 'rect',       // 改为方形
  huangong: 'triangle', // 改为三角形
  qiaoliang: 'diamond'  // 菱形
};
```

可选形状：`circle`（圆形）、`rect`（方形）、`roundRect`（圆角矩形）、`triangle`（三角形）、`diamond`（菱形）、`pin`（大头针）、`arrow`（箭头）

**修改尺寸：**
```javascript
symbolSize: [20, 20]  // 宽20px，高20px
```

**修改颜色：**
```javascript
itemStyle: {
  color: '#FF0000',        // 填充色
  borderColor: '#FFFFFF',  // 边框色
  borderWidth: 2           // 边框粗细
}
```

### 9.3 更换背景图片

1. 准备新背景图片（建议1920×1080以上，深色背景）
2. 替换文件：`images/detailed_images/background.png`
3. 或修改 topics.html 中的路径：

```html
<style>
  body {
    background: url('新图片路径.png') no-repeat center center fixed !important;
    background-size: cover !important;
  }
</style>
```

### 9.4 更换专题图标

1. 准备新图标（建议100×100像素，PNG格式，透明背景）
2. 替换对应文件：
   - `icon_minju.png` - 民居图标
   - `icon_guanfu.png` - 官府图标
   - `icon_huangong.png` - 皇宫图标
   - `icon_qiaoliang.png` - 桥梁图标
